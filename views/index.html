API Project: Exercise Tracker

This is my project of the Exercise Tracker challenge for the freeCodeCamp API and Microservice certification. It was built based on the boilerplate available here.

Create a New User

POST /api/exercise/new-user
New User

Add exercises

POST /api/exercise/add
Add Exercise

Get users's exercise log:

Just hit the endpoint below. Notice that the 'userId' query param is required, while the other ones are optional.

GET /api/exercise/log?{userId}[&from][&to][&limit]

// from, to = dates (YYYY-MM-DD); limit = number
